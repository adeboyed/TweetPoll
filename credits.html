TweetPoll

TweetPoll is for gathering the public's opinion on a particular topic using tweets.

Please note, we take no responsibility for the results produced by the output of this website. This website uses the latest public tweets to calculate its results and uses common machine learning algorithms to calculate opinion. Any apparent favouring of a person or entity due to the machine learning is purely accidental and not intentional.
